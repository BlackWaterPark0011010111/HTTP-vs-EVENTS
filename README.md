# HTTP-vs-EVENTS
Эта тема идет после понимания микросервисов.
http vs events, и как вообще я могу связать несколько сервисов?
у меня есть три сервиса:
    один отвечает за пользователей
    второй отвечает за заказы
    третий отвечает за баланс
и если наш гость делает заказ на нашем сайте,то все три должны быть связаны между собой. то есть,есть юзер,он делает заказ и баланс изменяется, то есть есть заказ,он принадлежит такому-то пользователю  и у этого пользователя должен как-то измениться его баланс.между нашим гостем(фронтенд) мы должны соединять все по HTTP,HTTP API.Но наши сервисы будут очень тесно связаны между собой.Ведь когда мы используем HTTP, это значит что у нас есть структура определенная,которую нам нужно использовать,например чтобы создать пользователя,нужно указать userid,email,password,...... .
